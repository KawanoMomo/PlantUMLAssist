# ADR カテゴリ

| カテゴリ | 説明 |
|---|---|
| **アーキテクチャ** | システム全体の構造、レイヤ分割、モジュール境界 |
| **インターフェース** | モジュール間API、契約、抽象化 |
| **インタラクション** | ユーザー操作モデル、UIメタファ、入力デバイス |
| **UI/UX** | レイアウト、視覚デザイン、操作の一貫性 |
| **テスト戦略** | テスト設計手法、ツール選定、カバレッジ目標 |
| **エージェント運用** | サブエージェント設計・連携・責務分割 |
| **配布・運用** | ビルド、リリース、互換性、ライセンス |
